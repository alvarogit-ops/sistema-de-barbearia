# Sistema de Barbearia

##  Visão Geral

O Sistema de Barbearia é um projeto acadêmico desenvolvido com o objetivo de simular o fluxo operacional de uma barbearia real, incluindo agendamento, gestão de clientes e organização de serviços.

O projeto é utilizado como base prática para aprendizado progressivo de desenvolvimento de software, evoluindo junto com o domínio de lógica de programação e tecnologias web.

---

##  Objetivo do Projeto

### Objetivo Geral
Desenvolver um MVP funcional de sistema de barbearia com arquitetura simples e evolutiva, permitindo expansão futura para um modelo mais completo (incluindo possível evolução para SaaS).

### Objetivos Específicos
- Simular regras de negócio de um ambiente real (barbearia)
- Praticar desenvolvimento incremental de software
- Aplicar versionamento com Git e GitHub
- Evoluir habilidades em backend e frontend progressivamente
- Construir um portfólio técnico baseado em projeto real

---

##  Abordagem de Desenvolvimento

O projeto segue uma abordagem incremental, onde cada funcionalidade é desenvolvida conforme o nível atual de conhecimento técnico.

O desenvolvimento é guiado por etapas:

- Modelagem inicial com fluxogramas (arquitetura de informação)
- Prototipação de interface (Figma)
- Implementação progressiva por funcionalidades
- Evolução técnica conforme aprendizado em Python e Django

---

##  Estado Atual do Projeto

- Fluxograma da arquitetura em desenvolvimento (FigJam)
- Protótipos de baixa e alta fidelidade em revisão (Figma)
- Estrutura conceitual do sistema em definição
- Retomada da implementação backend planejada após consolidação da base em lógica de programação
