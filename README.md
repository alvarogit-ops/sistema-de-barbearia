# Sistema de Barbearia 

Autor: [Álvaro Yuri](https://github.com/alvarogit-ops)

## Um breve contexto: 

Este projeto foi iniciado na fase de digitar código utilizando o Django, PostgreSQL, HTML5, CSS3, Python 3. 
Só que eu não fiz um mapeamento de requisitos e uma modelagem de dados, então acabei usando muita IA sem entender o que eu estava fazendo. 
Esse projeto foi remodelado para que cada funcionalidade fosse testada conforme meu domínio sobre lógica e arquitetura de software. Ao invés de PostgreSQL, usei SQLite que é mais simples porque antes tive que manter um servidor no ar usando a Render.

## Qual é a proposta desse sistema?

Este sistema tem como proposta simular um agendamento e gestão baseados em uma barbearia real, com a orientação do professor [Diego Cirilo](https://github.com/dvcirilo). 

## Objetivos Gerais:

Desenvolver um MVP estruturado e validado localmente,
focando na otimização de filas de clientes,
para que o sistema possua uma arquitetura sólida o suficiente para, no futuro,
ser escalado e transformado em um modelo SaaS (Software as a Service) comercial.

## Objetivos Específicos:

* Compreensão das regras de negócios reais de um estabelecimento, desde o fluxo de um cliente até o faturamento da barbearia
* Praticar a experiência de entregas de sprints em um ciclo de vida de um software
* Dominar o versionamento de um repositório com Git e Github
* Contribuir o projeto para meu crescimento profissional em backend e frontend, utilizando este projeto com documentação como um portfólio destacado.
